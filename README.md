# Real estate management service 
Real estate management service with technologies MongoDB Express React Node.js.

## Installation
To install, use the following command: `npm install`  

Create .env file with   
PRETTY_LOGGING  
CONNECT_URL_DB  
SALT  
PORT  
JWT_SECRET_CODE  
LOGIN_EMAIL  
PASSWORD_EMAIL  

## Usage  
To run service:  
`cd server`  
`npm run dev`
